## Profile Overview

Data Analyst focused on transforming raw data into **clear, reliable, and actionable insights**.

My work combines **data cleaning, data preparation, analysis, dashboard development, and business-oriented storytelling** to support better decisions and improve performance.

I am particularly interested in projects where data can create **real business value**, optimize processes, and make complex information easier to understand for both technical and non-technical audiences.

---

## Core Focus

- **Data Cleaning & Preparation**
- **Business Intelligence & Dashboard Development**
- **Financial & Performance Analysis**
- **Data-Driven Decision Making**
- **Business Process Optimization**
- **Insight Communication & Storytelling**

# Featured Projects

## Instacart Data & Business Intelligence Analysis

End-to-end analytics project built using **Python, SQL, and Power BI** to explore customer behavior, product performance, and purchasing patterns from transactional data.

### Highlights
- 3M+ orders analyzed
- 73% returning customer rate
- Executive KPI dashboard
- Customer behavior analysis
- Basket and purchasing pattern analysis

### Repository
[View Project](https://github.com/MorgilloEdoardo/instacart-business-intelligence)
